# Forge-SR

# Introduction

Forge-SR is a simple and lightweight software rendering engine written in C++. It is designed for my learning and exploration of computer graphics concepts. It is not intended for production use but trying to build the system with modern C++ features(C++23/modules/etc) and software engineering practices.

# Features

- Basic 3D rendering pipeline

# Build

you need to install the latest version of C++ compiler and xmake build system.


```bash
# build all targets
xmake build

# show all targets
xmake show -t

# run target
xmake run <target>
```
